-git checkout main
-git pull awesome main
-git checkout nhánh bị conflict
-git rebase main
Kiểm tra nhánh đang đứng:
git branch
nếu như đang đứng ở nhánh bất định chúng ta cần kiểm tra status:
-git status

sửa fide bị conflict giữ lại code đúng và bỏ đi code sai + phần xương cá 
sau khi sửa fide conflick xong
-git add -A
-git rebase --continue

-git push origin "tên nhánh" -f
